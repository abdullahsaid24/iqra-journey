
# Quran PDF

This directory should contain the Quran PDF file named `mushaf.pdf`.

You can download a standard Mushaf PDF from one of these sources:
- [King Fahad Quran Printing Complex](https://qurancomplex.gov.sa/kfgqpc-quran-hafs/)
- [Tanzil.net](https://tanzil.net/download/)
- [Quran.com](https://quran.com/download)

Download the PDF and place it in this directory with the name `mushaf.pdf`.
